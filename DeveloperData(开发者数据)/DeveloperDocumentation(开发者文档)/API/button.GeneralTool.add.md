# 参数
```
{
    "buttonImage": "string",
    "buttonName": "string",
    "callbackId": "string"
}
```
## buttonImage
需添加按钮的名称
## buttonName
需添加按钮的名称
## callbackId
按钮点击时触发事件的Id
# 功能
在一般工具中注册一个按钮
