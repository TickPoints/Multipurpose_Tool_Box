== 参数 ==
{
    "buttonName": "string",
    "buttonImage": "string",
    "callbackId": "string"
}
=== buttonName ===
需添加按钮的名称。
=== buttonImage ===
需添加按钮的图标路径。
=== callbackId ===
按钮点击时触发事件的Id。
== 功能 ==
在OP工具中注册一个按钮。
== 返回包 ==
Void
当出现bug等情况时，还会返回一个结果包(id: result)。
Tip: 虽无常规返回包，但在按钮点击时会触发事件返回包:
Id: "callback"
{
    "sourceName": "string"  // 点击按钮的玩家名
}
