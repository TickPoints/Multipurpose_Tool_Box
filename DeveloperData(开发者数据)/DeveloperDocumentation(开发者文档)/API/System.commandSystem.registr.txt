Tip: 使用了系统级权限
== 参数 ==
{
    "commands": "object"
}
=== commands ===
注册的一组命令，如下格式(具体见"../命令系统"或用SDK生成):
{
    "runGameCommand": {
        "type": "function",
        "run": function(performer, gameCommand) {
            try {
                return JSON.stringify(performer.runCommand(gameCommand));
            } catch {
                return JSON.stringify(performer.runCommand(gameCommand));
            }
        }.toString(),    // 将函数.toString()以正常传输
        "config": {
            "parameters": [{
                "type": "string"
            }],
            "RequiredPermission": 5,
            "needPermission": true
        }
    }
}
== 功能 ==
将一组命令注册到 commandSystem。
== 返回包 ==
当出现bug等情况时，还会返回一个结果包(id: result)。
Void
