== 参数 ==
{
    "RequestCode": "number"
}
=== RequestCode ===
请求码，只能为0或1，为0时，会在控制台打印"The test passed."，为1时则会返回包{"result":"The test passed."}。
== 功能 ==
一个测试，用于帮助插件确定多用工具包是否正常运行(或者是当前的基础协议是否正常)，如不正常则不会有效。
== 返回包 ==
Tip:可能为Void
当出现bug等情况时，还会返回一个结果包(id: result)。
Id: "testResult"
{
    "result": "string"  // "The test passed."
}
